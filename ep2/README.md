# Para executar o sistema:

1. Primeiro, clone o repositorio 
2. Segundo estabeleça a maquina virtual:
```console
virtualenv env
```
3. Em seguida, ative o ambiente virtual:
```console
env/Scripts/activate
```
4. Utilize o arquivo requirements .txt para instalar as dependencias
```console
pip install -r requirements.txt
```
5. Execute a aplicação (na pasta src/)
```console
python application.py, ou ative pelo visual studio code através do botão de play
```
